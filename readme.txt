This widget will show 1 most post with highest comment count by selected author and one rencent comment for same post.

------------------------------------------------
			   Description
------------------------------------------------

This widget checks database for post with highest comment count by selected author and one recent comment for same post and display on sidebar.

In admin section widget area, you can change author from dropdown.



------------------------------------------------
                    INSTALLATION
------------------------------------------------
Just upload the comment_widget folder in your plugin folder and activate from admin section.


------------------------------------------------
			     How to use
------------------------------------------------
After installation, in admin section, go to Appearance -> Widgets. You can find Comments Widget under Available Widgets section. Just click on it and and then click on Add Widget button. In widget, you will get a form to select an Author form dropdwon. Just select one and save. And that's all. This widget now will be displayed on side bar of pages.
